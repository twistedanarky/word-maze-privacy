# Word Maze: Word Search Puzzle - GitHub Pages Site

This repository hosts the Privacy Policy and Product Support pages for app store submission.

## Files

- `index.html` - Landing page
- `privacy.html` - Privacy Policy
- `support.html` - Product Support
- `styles.css` - Shared site styles

## Publish with GitHub Pages

1. Push this repository to GitHub.
2. Go to **Settings > Pages**.
3. Under **Build and deployment**, set:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (or your default branch)
   - **Folder:** `/ (root)`
4. Save settings.
5. Wait for deployment, then use the generated URL in App Store Connect and Google Play Console.

## Suggested App Store URLs

- Privacy Policy URL: `https://<your-github-username>.github.io/<repo-name>/privacy.html`
- Support URL: `https://<your-github-username>.github.io/<repo-name>/support.html`
